## 文件说明

- `index.html`：个人官网首页，包含结构化数据和 FAQ。
- `styles.css`：页面样式。
- `llms.txt`：给 AI 读取的简明身份说明。
- `robots.txt`：允许搜索引擎抓取。
- `sitemap.xml`：站点地图。
- `404.html`：访问错误链接时显示的页面。
- `.nojekyll`：兼容 GitHub Pages 的静态发布标记；使用其他平台时保留也不影响。
- `hosting-options-guide.md`：不同上线方式的选择说明。
- `github-pages-publish-guide.md`：把 GitHub Pages 作为低成本托管方式时的操作说明。
- `launch-and-coaching-checklist.md`：上线与后续纠偏清单。
- `domain-decision-sheet.md`：域名选择判断表。
- `ai-answer-test-log.csv`：上线后 AI 问答测试记录表。
- `decision-gates.md`：后续必须由用户确认后才能继续的决策关口。
